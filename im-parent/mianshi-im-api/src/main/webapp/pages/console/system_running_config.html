参数说明 参数值 变量名 参数说明
XMPP 心跳超时值 XMPPTimeout 此值为秒数，超过这个秒数未收到心跳则视为离线；同时重启通讯服务器才能生效
是否显示在线状态 isOpenOnlineStatus 默认为关闭，用于客户端是否显示在线状态
是否开启短信验证码注册 isOpenSMSCode 默认开启,只对注册有效
是否开启集群模式 isOpenCluster
HTTP是否安全验证 isAuthApi
手机号搜索用户 telephoneSearchUser
昵称搜索用户 nicknameSearchUser
使用手机号或者用户名注册 regeditPhoneOrName
用户昵称否词 userNicknameNotword 多个词之间用逗号分隔（全半角均可）。用于判别用户注册或修改昵称时是否包含该项设置的否词，包含则操作失败
注册邀请码 registerInviteCode
敏感词过滤 isKeyWord 同时必须重启通讯服务器才能生效，消息加密不能过滤,设计如此
保存单聊聊天记录 isSaveMsg 同时必须重启通讯服务器才能生效
保存群聊聊天记录 isSaveMucMsg 同时必须重启通讯服务器才能生效
消息延时删除时间（小时） msgDelayedDeleteTime 撤回，阅后即焚，删除
保存系统发送消息 isSaveSystemMsg 公众号发送的，后台发到群组的，压测的，同时必须重启通讯服务器才能生效
是否开启Google推送 isOpenGoogleFCM 谷歌推送开关控制（需服务器在大陆外启用有效）
短信服务支持 SMSType
图形验证码类型 imgVerificationCode
通讯录自动添加好友 isAutoAddressBook
是否保存接口请求日志 isSaveRequestLogs
直播赠送礼物分成比率 giftRatio
在线咨询链接 promotionUrl 示例：https://www.baidu.com/?pid=10000 pid后面的数字是你的用户ID,该ID很重要。
注册默认成为好友的用户手机号 defaultTelephones 支持填写多个带区号的手机号，多个带区号的手机号之间用“，”分隔
注册默认进群 defaultGroups 支持填写多个群ID，多个群ID之间用“，”分隔
隐私政策URL地址的前缀 privacyPolicyPrefix
创建公司是否需要管理员审核 createCompamyIsNeedCheck
用户主动加入公司是否需要创建者确认 userJoinCompanyIsNeedManagerConfirm
邀请加入公司是否需要用户确认 inviteJoinCompanyIsNeedUserConfirm
最大群人数 maxCrowdNumber 所有群最大群人数设置
是否开启视界评论 banComment 是否开启视界评论
资源数据库url resourceDatabaseUrl 资源数据库url
发布朋友圈是否审核 publishMsgWhetherCheck 发布朋友圈是否需要审核
发布短视频是否审核 publishMsgLableWhetherCheck 发布短视频是否需要审核
更新用户地理位置频率（单位:秒） locationModiyTime 更新每个用户地理位置的频率时间（单位:秒）
后台管理--短信验证码登录手机号 openAdminLoginCode 填写了手机号，登录后台管理则需要短信验证,不填写则关闭。当前填写的手机号为超级管理员(1000)的手机号
接口请求白名单 为空即不限制 requestApiList 多个之间用 “,” 分隔
第三方系统集成对接配置 查看
员工模板地址 employeeTemplateUrl 用于公司管理下载员工模板
部门模板地址 departmentTemplateUrl 用于公司管理下载部门模板
开启谷歌验证器 googleVerification 开启谷歌验证
开启客户端视频语音 videoVoice 开始客户端视频语音
用户注册和注册码加好友配置 registerInviteCodeSetting 用户注册和注册码加好友配置,配置示例 {"open":true,"upperToLowerMsg":"你好，朋友。","lowerToUpperMsg":"你也好啊"} 其中参数open的值true代表开启，false代表关闭,upperToLowerMsg:上级给下级发送的消息，lowerToUpperMsg：下级给上级发送消息，不配置则不发送。
